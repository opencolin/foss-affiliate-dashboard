Self-Hosting Setup Guide

Step-by-step complexity breakdown for each tool

Mautic Complex
StackPHP 8.0+, MySQL/MariaDB, Apache/Nginx, npm
Min RAM2 GB recommended
Shared HostingNot Supported
DockerAvailable
  1. Provision a VPS (2+ GB RAM) with Ubuntu/Debian
  2. Install PHP 8.0+ with extensions: xml, mysql, imap, zip, intl, curl, gd, mbstring, bcmath
  3. Install and configure MySQL/MariaDB
  4. Install Apache/Nginx and configure virtual host
  5. Install Composer and Node.js/npm
  6. Clone Mautic repo, run composer install
  7. Configure .env.local with DB + mailer settings
  8. Run php bin/console mautic:install
  9. Set up cron jobs for campaigns and email processing
  10. Configure SSL (Let's Encrypt recommended)
Affiliate note: No native affiliate layer — requires plugins or custom webhook/API integrations.
Raider Moderate
StackRust binary, MySQL
Min RAM256 MB (very lightweight)
Shared HostingNot Supported
DockerOfficial Image
  1. Pull official image: docker pull valeriansaliou/raider:v1.2.3
  2. Or install via Debian APT (Debian 12 Bookworm)
  3. Or via Cargo: cargo install raider-server
  4. Configure config.cfg — secret keys, DB URL, SMTP, branding
  5. Import raider.sql schema into MySQL
  6. Generate tokens: openssl rand -base64 32
  7. (Optional) Add Fixer API key for multi-currency
  8. Start: ./raider -c /path/to/config.cfg
  9. Integrate payment Track API into your backend
Integration note: Backend must POST payment events to the Track API on each conversion — a single HTTP call.
Weferral Easy–Moderate
StackNode.js, React, PostgreSQL
Min RAM512 MB
Shared HostingPossible (Node hosts)
DockerCommunity Only
  1. Clone frontend: git clone https://github.com/WeferralHq/weferral.git
  2. cd weferral && npm install && npm start
  3. Clone backend: git clone https://github.com/WeferralHq/weferral-api.git
  4. cd weferral-api && npm install && npm start
  5. Visit http://localhost:4100/setup to init DB + admin user
  6. Install Weferral Tracker JS snippet (separate repo)
  7. For production: npm run build:prod, deploy /dist
Maturity note: Explicitly marked "not production ready." Plugin framework and RBAC are listed as coming soon.
RefearnApp Easy
StackNode.js, Docker
Min RAM512 MB
Shared HostingVia Coolify
DockerOfficial Docker Setup
  1. Clone the RefearnApp repository from GitHub
  2. Copy and configure .env with Stripe/Paddle keys
  3. Run docker compose up -d to start all services
  4. Or deploy via Coolify with one-click from repo
  5. Add JS tracking snippet to your website
  6. Configure Stripe/Paddle webhooks to point at RefearnApp
  7. (Optional) Purchase Pro/Ultimate license key for team + branding features
License note: Core is free and open-source. Team management, custom branding, and coupon tracking require a one-time paid license key ($199–$299).
RefRef Moderate
StackNext.js, TypeScript, PostgreSQL, Redis, BullMQ
Min RAM1 GB (Redis + PostgreSQL overhead)
Shared HostingNot Supported
DockerAvailable
  1. Clone: git clone https://github.com/refrefhq/refref.git
  2. Install dependencies via pnpm install (Turborepo monorepo)
  3. Provision PostgreSQL and Redis instances
  4. Configure .env with DB, Redis, auth, and SMTP settings
  5. Run docker compose up or deploy monorepo services individually
  6. Add JS referral attribution snippet to your site
  7. Configure reward approval rules (manual or automatic)
Alpha note: RefRef is explicitly in alpha — expect bugs and breaking changes. No stable releases published yet. Best for early adopters and contributors.
